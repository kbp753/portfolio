Project Description

The Facial expression Detection from Photos project employs machine learning and artificial neural networks to analyze facial expressions in images and classify them into various emotional categories. This project aims to provide insights into human emotions based on visual data, achieving over 63% accuracy on a 7-class emotion dataset.

Features

  • Image Preprocessing: Converts raw images into grayscale, normalizes pixel values, and resizes images for uniformity.
  • Feature Extraction: Uses OpenCV to detect facial landmarks and extract relevant features for emotion classification.
  • Model Development:Builds and trains convolutional neural networks (CNNs) using TensorFlow and Keras to classify emotions
  • Data Augmentation: Applies techniques such as rotation, flipping, and scaling to increase the diversity of the training dataset and improve model generalization.
  • Model Evaluation:Utilizes accuracy, precision, recall, and F1 score to evaluate model performance
